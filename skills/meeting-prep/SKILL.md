# Meeting Prep

## Workflow (keep it light)
1. Read `vault-ops` skill and follow its conventions.
2. Gather only essential context:
   - person note (`Coupa/People/...` or equivalent)
   - historical 1:1 note(s) for that person (`Coupa/1-on-1s/<Person>/`)
   - latest 1-2 daily reports (`Daily/Reports/`)
   - explicit instructions from user message
3. Create note in `Inbox/` named:
   - `YYYY-MM-DD - <Person> - Meeting Prep`
4. Use template structure from `Templates/Meeting Prep`.

## Fill the template
### Prep Message
- Write a short message Spencer can paste directly.
- Voice: direct, warm, practical, low-fluff.
- Include:
  - 2-4 bullets for topics
  - 1 short context paragraph (why this matters now)

### Agenda
- 4-6 concise bullets, ordered by priority.

### My Objectives
- 3-6 outcome-focused bullets (what Spencer needs to leave with).

### Other Context
- Only high-signal reminders (risks, sensitivities, key people/dependencies).

## Constraints
- Keep the entire note brief.
- Prefer bullets over prose.
- Prioritize user’s explicit asks over inferred structure.
