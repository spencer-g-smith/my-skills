# Knowledge Base Governance

Use when saving articles, references, documents, or reading material.

## Capture-first path
1. Capture quickly in `Inbox/YYYY-MM-DD.md`.
2. Promote to canonical destination as soon as practical.

## Canonical destinations
- `Personal/Reading/Sources/` — raw source captures
- `Personal/Reading/Insights/` — distilled claim notes
- `Personal/Self-Development/` — personal application notes
- `Coupa/AI Roundup/` — work application / weekly engineering roundup drafts

## Routing rules
- `Inbox/` is the universal quick-drop for URLs and unstructured thoughts.
- Do not file raw reading captures into `Workshop/` unless directly tied to active project execution.
- Prefer linking over duplication; update relevant `_MOC.md` when navigation changes materially.

## Cadence
- Weekly ingestion/distillation process lives in `references/weekly-maintenance.md`.
